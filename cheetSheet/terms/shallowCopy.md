# シャローコピー（浅いコピー）

オブジェクトの「1 階層だけ」をコピーする方法

## 基本例

```js
const a = {
  name: "山田",
  age: 20,
};

const b = { ...a };

b.name = "田中";

console.log(a.name); // "山田"
```

プリミティブ値は独立する

## 注意

```js
const a = {
  user: {
    name: "山田",
  },
};

const b = { ...a };

b.user.name = "田中";

console.log(a.user.name); // "田中"
```

ネストはコピーされない

## なぜこうなる？

```text
a.user → メモリ
b.user → 同じメモリ
```

中身は共有される

## シャローコピーの特徴

- 1 階層だけコピー
- ネストは参照共有
- 軽い（高速）

## 作り方

### ● スプレッド構文（よく使う）

```js
const b = { ...a };
```

### ● Object.assign

```js
const b = Object.assign({}, a);
```

### ● 配列の場合

```js
const arr2 = [...arr1];
```

## 🆚 ディープコピーとの違い

| 項目           | シャローコピー | ディープコピー |
| -------------- | -------------- | -------------- |
| コピー範囲     | 1 階層         | 全部           |
| ネスト         | 共有           | 独立           |
| パフォーマンス | 高速           | 重い           |

## React での使い方（超重要）

```js
setState({
  ...state,
  name: "田中",
});
```

これが基本

## ネスト更新の正しいやり方

```js
setState((prev) => ({
  ...prev,
  user: {
    ...prev.user,
    name: "田中",
  },
}));
```

必要な部分だけコピー

- 自分のどういうところが強みになる武器かの言語化ができる

### すぐにできる実践方法

- スマートフォンのアプリや、日常的に利用している Web サービスの UI を意識して観察する

  例

  - 入力チェックひとつをとっても、入力中にバリデーションが行われる場合や、フォーカスアウト時に実行される場合、送信ボタンなどのアクション時にまとめてチェックされる場合がある
  - 検索機能では、検索ボタン押下で実行されるものや、入力途中で自動的に絞り込みが行われるものがある
  - レスポンシブデザインの観点から、どのような要素がどこに配置される傾向にあるかを意識する

- あえて意地の悪い操作を行い、どのようなエラーや挙動になるのかを確認する
- 普段利用しているサービスを観察し、「多くのサービスで共通している点」や「特定の種類のサービスに特有の挙動」について、自分なりに気づきを得るようにする

### 自走の定義とは

参考サイト : https://www.jmam.co.jp/hrm/column/0152-listening_ability.html

- 対面で向き合って話を聞く
- この人ならこれ喋ってもいいなという雰囲気を作る
- 部分部分で相手の理解度を伺う

  例）

  - ここまでの前提って xx であってますか？

### 言語化能力の鍛え方

中長期で熟成させる

- 自分でしっかり意見を持つという事の練習をする
  - 思っていることを書きだす
  - 人と会話して伝える
- 前提知識の連携をしっかり行う

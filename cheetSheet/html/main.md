# main

ページの「主役となるコンテンツ」をまとめるための要素

## main>タグとは

```html
<main>
  <h2>記事タイトル</h2>
  <p>ここがメインの内容です</p>
</main>
```

- ページ内で最も重要なコンテンツ領域
- 基本的に 1 ページに 1 つだけ使う
- ナビゲーションやサイドバーは含めない

## 何ができるのか

1.  記事やコンテンツの本体を置く

    ```html
    <main>
      <article>
        <h2>ブログ記事</h2>
        <p>本文...</p>
      </article>
    </main>
    ```

1.  商品一覧やサービス紹介

        ```html
        <main>
          <h2>商品一覧</h2>
          <div>商品A</div>
          <div>商品B</div>
        </main>
        ```

    ③ アプリのメイン画面

        ```html
        <main>
          <h2>ダッシュボード</h2>
          <p>データ表示など</p>
        </main>
        ```

## どこに使うのか

```html
<body>
  <header>ヘッダー</header>

  <main>メインコンテンツ</main>

  <footer>フッター</footer>
</body>
```

- これが基本のレイアウト

## 重要な考え方

- ページの「主役」だけを入れる

  - メニュー・広告・サイドバー ×
  - 記事・商品・コンテンツ本体 〇

- 基本は 1 つだけ

→ アクセシビリティ的にも 1 つが原則

## div との違い

タグ 意味

- div ただの箱
- main 主コンテンツという意味

→ スクリーンリーダーなどが理解しやすくなる

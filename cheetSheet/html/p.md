# p

段落（文章のまとまり）を表すための要素

## p>タグとは

```html
<p>これは段落です</p>
```

- 文章のひとかたまり（段落）を表す
- テキストを書くときの基本タグ
- 自動で上下に余白がつく（ブラウザのデフォルト）

## 何ができるのか

1. 文章を段落として表示

   ```html
   <p>これは1つ目の段落です。</p>
   <p>これは2つ目の段落です。</p>
   ```

1. 長い文章を読みやすく分割

   ```html
   <p>今日はいい天気です。</p>
   <p>散歩に行きたくなりますね。</p>
   ```

1. 他のタグと組み合わせて説明文を作る

   ```html
   <article>
     <h2>記事タイトル</h2>
     <p>この記事の内容です。</p>
   </article>
   ```

1. リンクや強調と組み合わせる

   ```html
   <p>詳しくは <a href="#">こちら</a> をご覧ください。</p>
   ```

## どこに使うのか

- テキストを書く場所すべて

  ```html
  <main>
    <p>本文テキスト</p>
  </main>
  ```

- article や section の中で使う

  ```html
  <section>
    <h2>説明</h2>
    <p>ここに説明を書く</p>
  </section>
  ```

## 重要な考え方（ここ大事）

- 「文章のまとまり」に使う

  - 改行のためだけに使う ×
  - 意味のある段落として使う ○

- 空タグとして使わない

  ```html
  <p></p>
  ```

  ← NG（意味がない）

- 入れ子にできない（超重要）

  ```html
  <p>
    <p>これはNG</p>
  </p>
  ```

  → p タグの中に p タグは入れられない

## div との違い

タグ 意味

- div ただの箱
- p 段落（文章のまとまり）

## br との違い

タグ 役割

- p 段落を分ける
- br 改行するだけ

例：

```html
<p>これは段落1です</p>
<p>これは段落2です</p>
```

```html
<p>
  これは文章です<br />
  改行されます
</p>
```

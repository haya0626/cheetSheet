# ul

順序なしリスト（箇条書き）を作るための要素

## ul タグとは

```html
<ul>
  <li>項目1</li>
  <li>項目2</li>
</ul>
```

- 箇条書きリストを作るタグ
- 中に`li`タグを入れて使う
- 順番に意味がないリストに使う

## 何ができるのか

1. 箇条書きリストを作る

   ```html
   <ul>
     <li>りんご</li>
     <li>みかん</li>
     <li>バナナ</li>
   </ul>
   ```

1. ナビゲーションメニュー

   ```html
   <nav>
     <ul>
       <li><a href="#">Home</a></li>
       <li><a href="#">About</a></li>
     </ul>
   </nav>
   ```

1. 入れ子（ネスト）で階層構造

   ```html
   <ul>
     <li>
       親
       <ul>
         <li>子1</li>
         <li>子2</li>
       </ul>
     </li>
   </ul>
   ```

1. カード一覧や UI のベース

   ```html
   <ul>
     <li>商品A</li>
     <li>商品B</li>
   </ul>
   ```

## どこに使うのか

- メニュー一覧

  ```html
  <ul>
    <li>メニュー1</li>
    <li>メニュー2</li>
  </ul>
  ```

- データ一覧・リスト表示

  ```html
  <section>
    <h2>一覧</h2>
    <ul>
      <li>項目1</li>
    </ul>
  </section>
  ```

## 重要な考え方（ここ大事）

- 「順番に意味がないリスト」に使う

  - 順番が重要 → ol を使う
  - 順番関係ない → ul を使う ○

- li とセットで使う

  ```html
  <ul>
    <li>OK</li>
  </ul>
  ```

  ```html
  <ul>
    テキスト ← NG
  </ul>
  ```

- 見た目ではなく意味で使う

  - レイアウト目的だけで使う ×
  - リストとして意味があるときに使う ○

## div との違い

タグ 意味

- div ただの箱
- ul リスト（まとまり）

## ol との違い

タグ 役割

- ul 順序なしリスト
- ol 順序ありリスト

例：

```html
<ul>
  <li>順不同</li>
</ul>

<ol>
  <li>手順1</li>
  <li>手順2</li>
</ol>
```

## よく使う属性

※ul 自体はあまり属性を持たないが、実務では重要

- class → スタイル指定（超重要）
- id → JS やアンカー用

例：

```html
<ul class="menu">
  <li>Home</li>
</ul>
```

## li 側でよく使う属性

- class → デザイン制御
- value（ol で使用）→ 番号制御

例：

```html
<ul>
  <li class="active">現在ページ</li>
</ul>
```

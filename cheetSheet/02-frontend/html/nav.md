# nav

ページ内やサイト内の「ナビゲーション（移動リンク）」をまとめるための要素

## nav タグとは

```html
<nav>
  <a href="/">ホーム</a>
  <a href="/about">概要</a>
</nav>
```

- リンク（`<a>`）をまとめる専用タグ
- メニューや目次などに使う
- HTML5 のセマンティックタグ

## 何ができるのか

nav タグでできること

1. グローバルメニュー（サイト上部）

   ```html
   <nav>
     <a href="#">Home</a>
     <a href="#">Blog</a>
     <a href="#">Contact</a>
   </nav>
   ```

1. サイドメニュー

   ```html
   <nav>
     <a href="#">カテゴリー1</a>
     <a href="#">カテゴリー2</a>
   </nav>
   ```

1. ページ内リンク（目次）
   ```html
   <nav>
     <a href="#section1">セクション1</a>
     <a href="#section2">セクション2</a>
   </nav>
   ```

## どこに使うのか

- ヘッダー内（よくある）

  ```html
  <header>
    <h1>サイト名</h1>
    <nav>メニュー</nav>
  </header>
  ```

- サイドバー

  ```html
  <aside>
    <nav>カテゴリ一覧</nav>
  </aside>
  ```

- フッター内

  ```html
  <footer>
    <nav>補助リンク</nav>
  </footer>
  ```

## 重要な考え方

- 「移動のためのリンク」だけに使う

  - ページ遷移・スクロール移動 ○
  - ただのリンク装飾やボタン ×

- 全てのリンクを入れるわけではない

  ```html
  <p>詳しくは <a href="#">こちら</a></p>
  ```

こういう普通のリンクは nav に入れない

## div との違い

タグ 意味

- div ただの箱
- nav ナビゲーション

→ スクリーンリーダーや SEO に重要

- 複数使って OK

  ```html
  <nav>メインメニュー</nav>
  <nav>フッターメニュー</nav>
  ```

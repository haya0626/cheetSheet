# ol

順序ありリスト（番号付きリスト）を作るための要素

## ol タグとは

```html
<ol>
  <li>項目1</li>
  <li>項目2</li>
</ol>
```

- 番号付きのリストを作るタグ
- 中に`li`タグを入れて使う
- 順番に意味があるリストに使う（超重要）

## 何ができるのか（重要まとめ）

1. 手順・ステップの表現

   ```html
   <ol>
     <li>ログインする</li>
     <li>商品を選ぶ</li>
     <li>購入する</li>
   </ol>
   ```

1. ランキング表示

   ```html
   <ol>
     <li>1位 商品A</li>
     <li>2位 商品B</li>
   </ol>
   ```

1. 入れ子（ネスト）で階層構造

   ```html
   <ol>
     <li>
       親
       <ol>
         <li>子1</li>
         <li>子2</li>
       </ol>
     </li>
   </ol>
   ```

1. UI のベース（リスト表示）

   ```html
   <ol>
     <li>ステップ1</li>
     <li>ステップ2</li>
   </ol>
   ```

## どこに使うのか

- 手順説明

  ```html
  <ol>
    <li>登録</li>
    <li>確認</li>
    <li>完了</li>
  </ol>
  ```

- 優先順位・ランキング

  ```html
  <ol>
    <li>重要</li>
    <li>普通</li>
  </ol>
  ```

## 重要な考え方

- 「順番に意味があるとき」に使う

  - 順番が重要 → ol ○
  - 順番関係ない → ul

- li とセットで使う（必須）

  ```html
  <ol>
    <li>OK</li>
  </ol>
  ```

- 見た目ではなく意味で使う

  - 数字を出したいだけで使う ×
  - 手順・順序に意味があるときに使う ○

## ul との違い

タグ 役割

- ul 順序なしリスト
- ol 順序ありリスト

## div との違い

タグ 意味

- div ただの箱
- ol 順序付きリスト

## よく使う属性（超重要）

### ● start（開始番号）

```html
<ol start="5">
  <li>項目</li>
</ol>
```

→ 5 から始まる

---

### ● reversed（逆順）

```html
<ol reversed>
  <li>3</li>
  <li>2</li>
  <li>1</li>
</ol>
```

---

### ● type（番号の種類）

```html
<ol type="A">
  <li>A</li>
  <li>B</li>
</ol>
```

指定できる値：

- "1" → 1,2,3（デフォルト）
- "A" → A,B,C
- "a" → a,b,c
- "I" → I,II,III
- "i" → i,ii,iii

## li 側でよく使う属性

### ● value（番号指定）

```html
<ol>
  <li value="10">10から開始</li>
</ol>
```
